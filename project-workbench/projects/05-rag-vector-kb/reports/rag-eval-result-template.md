## 失败归因分类

| 归因 | 说明 | 改进方向 |
| --- | --- | --- |
| 召回失败 | 正确 chunk 未被检索到 | 调整 embedding 模型或 chunk 策略 |
| 排序失败 | 正确 chunk 被检索到但排名靠后 | 调整重排策略 |
| 生成失败 | LLM 未正确使用上下文 | 调整 prompt 或模型 |
| 权限过滤 | 正确 chunk 被权限过滤 | 检查权限配置 |
| 来源缺失 | 文档未被索引 | 检查文档导入流程 |
