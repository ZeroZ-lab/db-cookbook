## 官方依据摘要

- Milvus 官方文档将 Milvus 定位为开源、高性能、可扩展的向量数据库，支持从本地原型到大规模分布式部署，并把非结构化数据、Embedding、向量集合、索引和搜索作为核心能力。
- Milvus 官方文档同时强调其部署形态、索引搜索算法、可扩展架构、监控、备份、CDC、连接器和 AI 工具集成，说明它是专门向量检索系统，而不是完整 AI 数据平台的全部。
- Qdrant 官方文档将 Qdrant 定位为 AI-native vector search / semantic search engine，并在用户手册中把 collection、vector、payload、indexing、quantization、multitenancy、search、filtering、hybrid queries 作为主要能力。
- Qdrant filtering 文档说明可以在搜索或检索 points 时对 payload 和 point id 设置条件，支持 AND、OR、NOT 等逻辑组合；这支持“过滤缩小检索范围”的表述，但不能等同为端到端权限治理。
- Qdrant hybrid queries 文档说明可以组合多个 query 或分阶段查询，例如结合 dense vector 和 sparse vector，通过融合方法合并结果；这支持“混合/多阶段检索”的表述。
